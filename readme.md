# dotenv

contains my personalized settings and scripts for Ubuntu

## included scripts
- Apps/dropbox-cloud-document

  Add Dropbox Cloud Document integration to xdg-open. Install it by running `Apps/dropbox-cloud-document/install.sh`

- Apps/scripts/terminal.sh

  Make sure that only 1 instance of terminal opens. Install it by set it as default terminal.

## required packages

will automatically installed when running `./install.sh`

- ohmyzsh
- xclip
- zsh
- git
- curl
